Recent Projects

Nafiza

Nafiza is a car rental management system designed to showcase object-oriented programming (OOP) principles. Developed collaboratively by myself, Zuha Noor, and Natasha Nassar, this Java application provides a user-friendly platform for car rentals. Key Features: 1.Extensive Car Catalog: Nafiza boasts a comprehensive catalog of vehicles, allowing users to browse and select a car that perfectly suits their needs. 2.Dynamic Rental Cost Calculation: The application calculates rental costs in real-time, factoring in user-specified criteria such as chosen vehicle, rental duration, mileage allowance, and optional insurance coverage. This ensures transparency and facilitates informed decision-making for the user. The Future Enhancements for the Project will include a connection to a database, user account system, advanced car catalog search.
Remedi

Remedi is a mobile application designed to empower users with a convenient and reliable platform for managing their medication schedules. Built by a team consisting of myself, Vaishak Ravindranath, and Arwa Vora, this Flutterflow Application focuses on user-friendliness and leveraging the capabilities of modern smartphones. Key Features: Streamlined Medication Management: Remedi offers a user-friendly interface for adding and organizing medication details, including medicine names, dosages, frequencies, and intake instructions. This comprehensive management system ensures all vital information is readily accessible. 2.Customizable Reminders: Remedi goes beyond simple alerts. Users can set up personalized reminders with flexible scheduling options, ensuring medication is taken at the right time and with the appropriate frequency. 3.Medication Tracking Remedi can optionally track medication history, offering insights into overall medication intake patterns. This information can be valuable for discussions with healthcare providers. Future Enhancements for Remedi, will include medication interaction checks, refill reminders, or educational resources about medication adherence.
